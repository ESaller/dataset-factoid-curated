Instructions

Our AI system generates a couple of answers for various quiz questions, and we need your help identifying which ones are correct. We have some "answer patterns", but sometimes they are out of date and imperfect. Nevertheless, they can help you identify the correct answer, so we show them next to the question; sorry if they are not very human readable sometimes.

Please read each question and select all answers which are correct based on the hint, the supporting sentences provided for each question, your knowledge and/or a quick Google search.

  • Multiple answers may be correct! Please select all. For example, the south-western tip of Engliand is Land's End in Cornwall; the hint is Cornwall, but also select the Land's End.
  • We prefer short, to-the-point answers. When asking about snack flavor, banana is better than originally a snack cake filled with banana cream.
  • Sometimes, no provided answer is correct. Maybe you know the right answer, otherwise please try to briefly check the answer using Google. Enter the correct answer in the provided text box.
  • Sometimes, you find multiple correct answers that were not provided. For example multiple years for an invention, or mountain height in different units. Please enter all of them, separated by the "|" character.
  • If you did not manage to find the answer, please tick "unknown". If you think the question has no correct or good answer, please tick "bad question".
